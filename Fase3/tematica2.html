CONCEPTOS BASICOS CSS3

CSS significa Cascading Style Sheets (Hoja de Estilos en Cascada). El lenguaje CSS describe cómo se mostrarán los elementos HTML en la pantalla. El lenguaje CSS ahorra mucho trabajo, ya que permite controlar el diseño de varias páginas web a la vez. Todas las hojas de estilo externas se almacenan en archivos CSS.
